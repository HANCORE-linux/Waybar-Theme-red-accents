## Theme for Waybar

 with little $${\color{#ff9b71} red}$$ light accents

## Installation

Install dependencies for cava bar (using Arch-Linux) and wttrbar (weather module)

```bash
  sudo pacman -S cava
```
```bash
  sudo pacman -S waybar-cava
```
```bash
  sudo pacman -S wttrbar
```
